About

I truly believe that everyone has something to teach me, So if you feel you have something I can learn from you, I'd love to connect or hear from you.

I program in Bash, Python, C, Javascript, HTML, CSS and SQL using them to ethically hack, experiment and develop software that matters.

In addition to this, I am building and maintaining my own home server to host files, hold containers for game servers, webservers and a local smart assistant.
